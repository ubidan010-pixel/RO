wwise project is stored here:
-----------------------------
//root/data/Sound/WwiseProject/project.wproj


generated files:
------------------
- //root/data/Sound/WwiseLUA/wwise_constants.ilu => def enum used in the rest of the project
- //root/data/Sound/WwiseLUA/wwise_lookuptable.ilu => lookup table : name <-> ID <-> GUID for all Wwise elements
- //root/data/Sound/WwisePackage/*.pck => folder containing packages generated by Wwise (contains sounds, streams, events, ...)

!!!-- For users lambdas --!!!
-> A robot via //root/data/Sound/WwiseProject/bat/generatePackages.bat will generate those files as soon as the content
to //root/data/Sound/WwiseProject/ is amended by archiving
-> Users retrieve P4 therefore a coherent whole


!!!-- The sound designers --!!!
-> They hide in their P4 workspace : //root/data/Sound/WwiseLUA/... and //root/data/Sound/WwisePackage/...
-> They generate the contents of these directories via generation banks in Wwise (via batches of pre / post processing)



adaptation of existing datas UAF:
---------------------------------
- //root/data/EngineData/GameConfig/soundconfig.isg
SoundConfig_Template struct contains 3 new structures
+ WwiseLookUpTable => correlation table name <-> ID <-> GUID. This is the content of wwise_lookuptable.ilu
+ WwiseBankList => list of banks common to all maps in the game (never unloaded)
+ WwiseEngineEventList => list of reference to events Wwise used directly by the engine (pause all, resume all, stop all ...)
+ "microZoffset = 0.0" => adding an offset between the microphone and the camera (useless ??)

- //root/data/EngineData/Helpers/sound_base.ilu
includeReference ("Sound/WwiseLUA/wwise_constants.ilu") for the enum defined by wwise_constants.ilu

- //root/data/Engine/Misc/Components/SoundComponent.ilu
Adding members in the structure "SoundComponent_Template"
+ WwiseBankList => list of bank charge by the component
+ WwiseInputList => list of input Wwise. exe: PSTN switch, ...


You must remove all references to RAKI structures that do not make sense with Wwise. Some
these structures have no effect and will be just ignored, however, those that are created by factories
should not be used as factories RAKI no longer exist. So you need to delete:
- EventSoundCommand
- EventSetReverbOnAux
- EventSetBusPitch
- EventSetBusFilter
- EventSetBusVolume
- EventSetBusAuxFactor
- EventStopMusic
- EventPlayMusic
- EventMetronomeSetBPM
- EventBusMix
- MusicComponent

- Triggers => not necessary because available via events
- "Dynamic dialogue" / "Arguments" => not useful for now
- Reload *. Pck => not possible + limited interest (reloading bank is enough)
- Tweak volumes on "ITF :: Actors"
